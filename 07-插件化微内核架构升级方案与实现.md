# 插件化微内核架构升级方案与实现

## 1. 微内核架构
内核只保留：权限、Schema、渲染引擎、插件系统。
其余能力（表单组件、页面组件、工作流、报表、大屏、数据源、图表、规则引擎）全部通过插件扩展。

## 2. 插件SDK
- 后端：`MicroKernelPlugin` + `PluginDescriptor` + `PluginContext`
- 前端：`FrontPlugin` + `FrontPluginDescriptor` + `PluginContext`

## 3. 插件接口
- install/start/stop/reload/uninstall 生命周期接口
- descriptor 提供依赖、权限、菜单、路由、配置

## 4. 插件注册中心
- 后端：`PluginRegistry`（注册、状态、依赖检查）
- 前端：`plugin-registry.ts`（注册、状态管理）

## 5. 插件加载器
- 后端：`PluginLoader`（加载、停止、热重载、卸载）
- 前端：`plugin-loader.ts`（load/reload/stop）

## 6. 前端插件机制
- `plugin-runtime.ts` 统一启动插件
- `config-center.ts` 统一插件配置中心
- 支持组件动态注册与路由/菜单扩展（descriptor）

## 7. Java完整实现
- `backend/src/main/java/com/lowcode/plugin/*`
- `PluginController` 提供插件管理 API

## 8. Vue完整实现
- `frontend/src/platform/plugin/*`
- 可与设计器/渲染器组合形成插件化扩展链路
