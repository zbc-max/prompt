# 企业级代码生成器设计与实现

## 1. 生成器架构

- 元数据输入层：读取表结构/模型配置（单表、主子表、树结构）
- 模板渲染层：Freemarker 模板引擎
- 产物编排层：后端 Java + MyBatis + API + 前端 Vue + SQL
- 在线服务层：预览接口、ZIP下载接口
- 扩展层：custom 模板目录覆盖默认模板

## 2. 模板结构

```text
backend/src/main/resources/templates
├── java
│   ├── controller.ftl
│   ├── service.ftl
│   └── mapper.ftl
├── vue
│   ├── index.ftl
│   ├── detail.ftl
│   └── tree.ftl
├── sql
│   └── schema.ftl
└── custom
    └── README.md
```

## 3. Velocity/Freemarker实现

当前实现采用 Freemarker（可平滑替换 Velocity）。核心类：
- `TemplateEngineService`：加载模板并渲染文本
- `CodeGeneratorService`：按场景组装文件清单

## 4. Java代码

关键文件：
- `model/GenModels.java`：元数据模型（项目、表、字段、上下文）
- `service/CodeGeneratorService.java`：单表/主子表/树结构生成编排
- `controller/GeneratorController.java`：在线预览与下载 API

## 5. Vue模板

- `templates/vue/index.ftl`：列表页（带权限按钮）
- `templates/vue/detail.ftl`：主子表详情页模板
- `templates/vue/tree.ftl`：树结构页面模板

## 6. ZIP生成

- `ZipUtil.zip(Map<String,String>)` 将多文件写入内存 ZIP
- `POST /api/gen/zip` 直接输出下载流

## 7. 在线预览

- `POST /api/gen/preview` 返回 `Map<文件路径, 代码文本>`
- 前端可直接做“文件树 + 代码编辑器”预览
